# Kerala Emergency Finder

Kerala Emergency Finder is a production-ready React + Vite application for discovering nearby emergency services across Kerala using live OpenStreetMap, Overpass, Open-Meteo, and KSDMA data.

## Features

- Automatic geolocation with graceful permission handling
- Nearby hospitals, police stations, fire stations, pharmacies, and blood banks
- Radius switcher for 5, 10, 20, 30, and 50 km
- Kerala-wide search by facility, district, town, village, or landmark
- Interactive Leaflet map with clustering, current location, and popups
- Emergency numbers, weather snapshot, and Kerala disaster alerts
- React Query caching, reusable UI, loading skeletons, and error boundaries
- Light and dark themes with premium glassmorphism styling
